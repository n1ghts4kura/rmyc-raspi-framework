  - 能够在安全范围内控制 pitch/yaw，实现基础瞄准与跟踪。
- 约束条件（占位）：
  - pitch 角度有限制；yaw 支持多圈旋转但单次命令有幅度限制。

- 角度表示方式（绝对/相对）、单位与归一化策略。

## 3. 360° 旋转能力的演进（占位）

- 从基础 yaw 控制到多步分段旋转的设计过程。

- pitch 机械极限与软件限幅的关系。
