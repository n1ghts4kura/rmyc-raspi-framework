## 1. 背景与目标

- 初始目标（占位）：
  - 在树莓派上通过 ONNX Runtime 跑通 YOLO 检测，并满足自瞄系统所需的帧率与延迟要求。
- 约束条件（占位）：
  - 仅在树莓派 Linux 上运行；开发机只做编辑与静态检查。

## 2. 初始版本设计（占位）

- recognizer 的基本结构、线程模型与与主控之间的接口。
- 模型选择与输入尺寸（例如 480x320）的大致考虑。

## 3. 关键迭代节点（占位）

- 性能相关迭代（模型精度 vs. FPS 的权衡）。
- 与自瞄、技能系统对接时的协议/数据结构演进。

## 4. 踩坑与经验（占位）

- 在树莓派上部署 ONNX 模型时可能遇到的问题。
- 摄像头分辨率、颜色空间、对焦/曝光等与识别效果的相互影响。

## 5. 未来规划（占位）
